# .NET8 医保结算模拟接口服务
## 项目介绍
市面稀缺C#医保接口实战Demo，对标医院HIS医保前置结算业务，求职医疗软件公司核心竞争力项目。
## 技术栈
.NET 8 ASP.NET Core WebApi、EF Core 8、MySQL、XML报文处理、日志框架
## 核心功能
1. 基于.NET8 + Nacos 搭建标准微服务架构，实现服务自动注册发现、配置热更新，彻底摒弃硬编码地址。
2. YARP 网关统一入口动态路由，自动感知服务上下线，实现三地医保服务统一流量治理。
3. 多服务完全故障隔离，基于 Polly 实现独立重试、超时、熔断，单地市服务异常不影响整体业务。
4. 采用 MySQL+EF Core CodeFirst 开发，自动化建库建表，数据结构规范、可迁移、可一键恢复。
5. 双层服务健康监控机制，结合 Nacos 心跳 + 业务自定义心跳入库，精准识别服务在线状态。
6. 自研可视化监控面板，实现地市服务红绿状态灯、调用统计、异常日志查询，运维直观。
7. 全链路日志持久化，Serilog 实现文件 + 数据库双存档，请求、异常、堆栈信息可完整溯源。
8. 稀缺差异化项目，市面极少 C# 医保结算微服务 Demo，适配医院、医保政务招标项目场景。
## 运行环境
Trae CN AI开发IDE
JDK 11、.NET 8 SDK、MySQL 8.0、Nacos 2.3.2
